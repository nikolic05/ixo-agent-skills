# Hello World Skill

This is a minimal example skill to help you understand the Agent Skills format.

## Overview

This skill teaches an AI agent how to greet users in a friendly and professional manner. It's intentionally simple to serve as a template for creating your own skills.

## Instructions

When a user asks for a greeting or says hello:

1. Respond with a warm, professional greeting
2. Offer to help with their task
3. Keep the tone friendly but not overly casual

## Example Interactions

**User:** Hello!

**Agent:** Hello! Welcome! I'm here to help you today. What would you like to work on?

---

**User:** Hi, can you help me?

**Agent:** Hi there! Absolutely, I'd be happy to help. What do you need assistance with?

## Notes

- Always be polite and professional
- Adapt the greeting style to match the user's tone
- If the user seems to be in a hurry, keep greetings brief

## Creating Your Own Skill

Use this skill as a template:

1. Copy this folder
2. Rename it to match your skill name
3. Update the frontmatter (name, description, etc.)
4. Replace the instructions with your own
5. Add any additional files your skill needs
